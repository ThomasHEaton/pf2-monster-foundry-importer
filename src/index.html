PF2e Foundry Monster Image Importer

Version 0.6.0 Compatible

Step 1: Select the pathfinder-bestiary.db inside of the Foundry App Data (typically AppData/Local/Foundry/Data/systems/pf2e/packs/pathfinder-bestiary.db.



Step 2: Select the image directory path. The files must be named with IDs to match the Nethys Monster IDs. i.e. https://2e.aonprd.com/Monsters.aspx?ID=48 would map to 48.png (Grizzly Bear)



Step 3: Copy your Image Directory to AppData/Foundry/systems/pf2e/monsters/


Step 4: Generate new Foundry DB. Download the file and replace your old pathfinder-bestiary.db. Restart Foundry and enjoy!
